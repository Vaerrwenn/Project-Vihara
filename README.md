# Project Pencatatan Keuangan Vihara
Web Application that could save transaction details that happened in Vihara

## Requirements
* Ruby 2.6.5
* Rails 6.0.2.2
* PostgreSQL 11.3
* Latest NodeJS 
* Latest Yarn
